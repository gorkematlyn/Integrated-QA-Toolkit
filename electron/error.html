Uygulama Yüklenemiyor

React uygulamasını yükleme sırasında bir problem oluştu. React derlemesi uygun konumda bulunamadı.

Lütfen aşağıdaki çözümleri deneyin:

  • Uygulamayı yeniden başlatın
  • "npm run build" komutunu çalıştırarak React uygulamasını yeniden derleyin
  • "node portable-build.js" komutu ile Windows paketini yeniden oluşturun
